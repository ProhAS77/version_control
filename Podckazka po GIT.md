## Подсказка по GIT

**Содание репозитория**
```sh
git init
```
*Проиндексировать файл*
```sh
git add
```
Закомитить файл и добавить комментарий
```sh
git commit -m "message"
```
## Работа с версиями

*Вывод версий*
```sh
Полное описание изменений:
git log

Отражение изменений в одну строку:
git log --oneline
```
*Переход между версиями*
```sh
git checkout <Номер версии>
```
